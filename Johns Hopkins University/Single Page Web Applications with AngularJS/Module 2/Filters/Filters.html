Because it's a lot more fun when we can interact, isn't it?

So, I added buttons to change the format of our already well known text, so you can choose three formats:

  1. The original one;
  2. Another with everything written in uppercase letters; and
  3. The final option, that allows you to show the text written in all lowercase letters.

The functions actuate in a manner that the input text that is shown doesn't get altered by the code, formating the input text prior to it's exhibition on the page.
