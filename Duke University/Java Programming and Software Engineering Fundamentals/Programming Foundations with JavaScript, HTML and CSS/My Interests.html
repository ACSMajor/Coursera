While I was searching at Wikimedia Commons web site for an image to include in my personal interests page, I found the below image, that was chosen as image of the day for 10th January 2024.

Since I watched recently an anime named Pluto at Netflix, I decided to use this first image that were presented to me as soon as I accessed Wikimedia Commons web site, just because one of the robots in the previously mentioned anime were named Mont Blanc.

At Wikimedia Commons I found an option to use this file on the web directly, so I decided to use this option.

By doing so, I was presented with a few options, two of them were most important to me:

  1. An option to select the image width in pixels (being the 4096px the widest option available); and
  2. A text area titled Embed this file, that I used to get the link with the appropriate credits and a reference to the license already provided.

For the extra challenge, I just changed the value provided for the width attribute of the HTML <img> Tag, inserting a percent value o 100%, that relates to the width of the viewport of the browser.

By doing so, the image has it's width automatically adjusted to fill the width of the browser's viewport.

If you want to know more about the HTML <img> tag and it's width attribute, just follow the above links to the W3Schools web site.
